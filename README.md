# 🍱 LeftoverChef — "What Can I Cook?"

Enter the ingredients you have at home and the app suggests recipes you can actually make. Save your favorites for later.

## Tech
- Frontend: HTML / CSS / vanilla JS
- Backend: Node.js + Express
- Database: JSON files (`data/recipes.json`, `data/favorites.json`)

## Run it in VS Code

1. Open this folder in VS Code.
2. Open a terminal (Terminal → New Terminal) and run:

```bash
npm install
npm start
```

3. Open your browser at **http://localhost:3000**

That's it. Add ingredients, click **Find recipes**, and save the ones you like.
